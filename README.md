# My Git Cheat Sheet
  
### Basic Commands:
    > git init
    > git clone <link / path>
    > git status
    > git fetch
    > git add <. / * / file_name>
    > git commit -m <"Commit Message">
    > git pull
    > git log
    > git remote -v
    > git remote add origin <link / path>
    > git push origin master
    > git checkout -b <branch_name>
    > git branch 
    > git merge <branch_name>
    > git branch -d <branch_name>
    > git reset
